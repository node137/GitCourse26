# Git-kurssin harjoitusprojekti

**Kurssi:** Git-versionhallinta - SOF013AS2A-3002
**Tekijä:** Noora Hänninen

## Repositorion sisältö
Tämä repositorio on luotu Git-opintojakson harjoitustöitä varten. Projekti sisältää:
- **Hello-sovellus:** Yksinkertainen HTML/CSS-sivusto.
- **Kello-toiminto:** JavaScriptillä toteutettu jatkuva ajan näyttö, joka on kehitetty yhteistyössä virtuaalisen tiimikaverin kanssa.
- **Harjoitushistoria:** Repositoriossa on käytetty useita haaroja (`master`, `develop`, `uusi-ominaisuus`) ja tageja harjoitus2 - harjoitus7 harjoitusten vaiheiden merkitsemiseen.
